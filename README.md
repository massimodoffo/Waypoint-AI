# Waypoint-AI

Waypoint AI is an intelligent travel planning assistant that helps users discover destinations, build itineraries, and get personalized recommendations. The app combines AI-driven suggestions with a clean, card-based interface to make trip planning fast and intuitive.

## Features

- AI-powered destination recommendations
- Interactive travel cards with destination details
- Dynamic itinerary building
- Responsive design for desktop and mobile

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- Netlify for deployment and serverless functions
